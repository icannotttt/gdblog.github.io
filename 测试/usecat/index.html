记录一些linux文本需要的指令

查看特殊字符：
cat -A filename

去掉文件中的M，这个^M用ctrl+V和ctrl+M打出来 ：
sed -i ‘s/^M//g’ filename

去掉文件的前三个字符：
sed -i ‘s/^…//g’ filename

提取文件第2行第3量作为变量：
third_column=$(awk ‘NR==2 {print 3}' data.txt) third_column=(cut -d’ ’ -f3 data.txt)

特殊分隔符分割的：
n=2 # 指定行号
delimiter=";" # 指定分隔符
third_column=(awk−vn="(awk -v n="(awk−vn="n" -v FS="$delimiter" ‘NR==n {print $3}’ data.txt)

使用cat和EOF输出变量到文件：
cat < output.txt
Column 3 value: $third_column
EOF

bc管道计算的时候小数部分不输出导致格式识别错误，加一个适配格式的0即可：
result=$(echo “scale=2; 10 / 3+0.00” | bc)
